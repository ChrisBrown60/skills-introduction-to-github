## 🌟 What You'll Learn

In this exercise, you'll practice the GitHub basics:

- Repositories
- Branches
- Commits
- Pull requests
- Merges

## 🙋 Who Is This For

This exercise is for anyone who is new to GitHub or wants a quick, guided introduction to the core workflow.

## 🛠️ What You'll Do

You'll complete a short hands-on workflow by:

1. Creating a branch
2. Committing a new file
3. Opening a pull request
4. Merging your pull request

## ▶️ How to Start

1. Open the repository's [Issues](../../issues) tab.
   1. Look for the exercise issue Mona created for you.
   2. Open that issue in a new tab so you can follow the instructions step by step.
2. Return to the **Code** tab in this repository.
   1. Complete each activity from the exercise issue in order.
   2. Watch for Mona's feedback after each step.

## 🎉 You're on your way!

Once you open your exercise issue, you'll be ready to begin your first GitHub workflow. Have fun!
